# Proyecto para aprender vue-cli y uso de github
Se trata de intentar optimizar el flujo de trabajo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
